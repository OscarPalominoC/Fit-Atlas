# FitAtlas Frontend Interface

The visual layer of the FitAtlas ecosystem. A high-fidelity, RPG-inspired dashboard designed for maximum operational immersion.

## 🎨 UI/UX Philosophy

- **Glassmorphism**: Layered transparency and blur for a premium, futuristic feel.
- **Neural Mapping**: Real-time visualization of muscle fatigue and training focus.
- **Dynamic Feedback**: Micro-animations and transitions powered by Framer Motion.
- **Offline Readiness**: Local asset resolution for animations and exercise metadata.

## 🛠 Technology

- **React 19**: Leveraging the latest features for state management and rendering.
- **Vite**: Ultra-fast build tool and dev server.
- **TanStack Query**: Robust data fetching and caching.
- **Tailwind CSS 4**: Next-generation utility-first styling.

## 📂 Key Components

- `src/components/MuscleMap.tsx`: The interactive neural atlas.
- `src/components/Analytics.tsx`: Biomechanical performance data visualization.
- `src/components/ExerciseModal.tsx`: High-fidelity exercise detail viewer (via Portals).
- `src/components/WorkoutLive.tsx`: Real-time session tracking interface.

## 🚦 Execution

1. **Install Packages**:
   ```bash
   npm install
   ```

2. **Run Dev Server**:
   ```bash
   npm run dev
   ```

3. **Build Production**:
   ```bash
   npm run build
   ```
